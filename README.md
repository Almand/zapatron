# Zapatron
#### Shoes Manager 2014

##### Instalacion

1. Clonar el repositorio

  `git clone https://github.com/rivfader/zapatron.git`

2. Asegurarse de tener instalado *bundle*, y cambiarse a la carpeta e instalar
   las gemas

   ```
   cd zapatron
   bundle install
   ```

3. Iniciar el servidor

  `rackup `
